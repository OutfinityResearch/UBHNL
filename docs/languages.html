1) Lexicon (the vocabulary)

The vocabulary defines types and allowed symbols. The rule is simple: every symbol used in CNL/DSL must be declared; otherwise it is a load error. Preferred source of vocabulary is CNL domain files (*.cnl). A JSON lexicon is legacy/optional for tooling.

Legacy JSON (optional, sketch)

{
  "domains": {
    "Cell": ["c0", "c1"],
    "Person": ["p0"]
  },
  "predicates": {
    "GeneA": { "arity": 1, "args": ["Cell"] },
    "ProteinP": { "arity": 1, "args": ["Cell"] },
    "HasFever": { "arity": 1, "args": ["Person"] }
  },
  "cnlPatterns": {
    "has Fever": "HasFever $1",
    "is Parent of": "Parent $1 $2"
  }
}

2) CNL (Controlled Natural Language)

CNL is “almost” natural language, but controlled: explicit quantifiers, explicit connectives, and clear scoping. Ambiguity is rejected, not guessed.

Examples

For any Cell c:
    If c has GeneA and c does not have Inhibitor then c has ProteinP.

Which Person p has Fever?

Determinism rules

  • All quantifiers must include a domain: For any Cell c: ....
  • Unknown symbol (predicate/constant/domain) = error.
  • Strict typing: predicate arity and argument types must match the lexicon.

3) DSL (authoring + canonical IR)

DSL is the canonical .sys2 format: compact, easy to process, good for theory files, and reproducible. It is built on the @ / $ / vocab rule.

The @ / $ / vocab rule

FormMeaningPolicy
@name the statement target (declaration / fact statement) at most one @ per line; must be the first token
$name working-memory reference in scope (binder or named expression) if it is not in scope → error
name vocabulary lookup (KB names + lexicon) if it does not exist → error

Statements

  • KB declaration: @Alice:Alice __Atom (persistent name)
  • Fact statement: @f1 HasFever Alice
  • Logic block: @rule1 ForAll Person graph p ... return $imp end
  • Witness query: @q Exists Person graph p ... return $expr end

Typical errors (intentional)

# invalid: two @ tokens on the same line
@p0 likes @p1
# fix: p1 must be $p1 (variable) or a vocabulary constant (bare p1)

# invalid: unknown vocabulary symbol
@f1 HasFlu Bob
# if "HasFlu" or "Bob" is not declared → load error

Straturi

  1. Trusted core: UBH IR + checkers (witness/certificate)
  2. Front-ends: CNL/DSL → typed semantic IR (determinist)
  3. Fragments: clasifică constrângerile și goal kinds
  4. Backends: SAT/SMT/QBF/MC/… cu artefacte verificabile
  5. Orchestrator: planificare, tactici (CEGAR), caching, explainability

Flux (high-level)

theory files (DSL/CNL) + session learn
           │
           ▼
parse/type (strict vocab) → typed semantic IR
           │
           ▼
fragment detection + goal kind
           │
           ▼
planner builds plan DAG (tactics/backends)
           │
           ▼
backend result + certificate/witness
           │
           ▼
checker validates → explain (origins) → user output

Architecture Overview

UBHNL is a universal reasoning system with a minimal trusted core (UBH) and pluggable backends (SAT/SMT/QBF/model checking/etc.) orchestrated via fragments and tactics.

Layers

  1. Trusted core: UBH IR + checkers (witness/certificates)
  2. Front-ends: CNL/DSL → typed semantic IR
  3. Fragments: classify constraint classes and goal kinds
  4. Backends: SAT/SMT/QBF/IC3/… with checkable artifacts
  5. Orchestrator: planning, tactics (CEGAR), caching, explainability

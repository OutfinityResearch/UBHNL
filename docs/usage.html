Quick Start (conceptual)

  1. Create a lexicon (vocabulary + types) as JSON.
  2. Write a theory in *.dsl (or *.cnl).
  3. Start a session, load theory files, then run query.
  4. Get a result + witness/certificate + explanation (via origins).

Target CLI (sketch)

# create a session and load lexicon
ubh session new --lexicon lexicon.json

# load theories (long-term memory)
ubh learn --file bio.dsl
ubh learn --file rules.cnl

# query: prove/refute/satisfy/count/optimize
ubh query --prove "proteinP(c0)"
ubh query --satisfy "exists ?c in Cell: proteinP(?c)"
ubh query --explain last

Target JS API (sketch)

import { createSession } from "ubh";

const session = await createSession({ lexiconPath: "lexicon.json" });
await session.loadTheoryFile("bio.dsl");

const result = await session.query({
  kind: "Prove",
  input: "proteinP(c0)"
});

console.log(result.status);
console.log(await session.explain(result, { style: "CNL" }));

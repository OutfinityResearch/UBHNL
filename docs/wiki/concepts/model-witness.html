Model / Witness

Ce înseamnă “SAT” într-un sistem care acceptă doar rezultate verificabile.

Definiție

Un model (a.k.a. witness) este o atribuire care face o teorie satisfiabilă. În UBH, un model dă valori pentru variabilele de intrare (și, în practică, pentru wire-urile relevante), astfel încât toate constrângerile să fie adevărate.

De ce “witness” este obligatoriu

  • SAT trebuie să fie checkable: verifici determinist că toate constrângerile sunt satisfăcute sub model.
  • Fără witness, “SAT” ar fi o afirmație neverificabilă → inacceptabil pentru audit.

Witness și holes (?x)

Pentru query-uri cu holes, witness-ul include și o asignare pentru ?x (de ex. o constantă din domeniu). Este tot un “model”, dar cu componentă “returnabilă”.

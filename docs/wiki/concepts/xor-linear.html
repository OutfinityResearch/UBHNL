XOR linear (ecuații GF(2))

Fragmentul liniar al UBH: ecuații de forma x1 ⊕ x2 ⊕ ... = b, rezolvate cu eliminare Gauss.

DRAFT: pagină de Wiki.
Wiki GF(2) SAT DS-002

Definiție

O ecuație XOR-liniară are forma: 

x1 ⊕ x2 ⊕ ... ⊕ xk = b   (b ∈ {0,1})

Este “liniară” în sensul algebrei peste GF(2). Astfel de sisteme se rezolvă eficient cu eliminare Gauss.

Ce intră aici în UBH

  • Definiții de poartă XOR: z = xor(a,b) devine z ⊕ a ⊕ b = 0.
  • ASSERT_EQ(a,b) devine a ⊕ b = 0.
  • ASSERT0/ASSERT1 sunt unit equations: w=0 / w=1.

De ce nu encodezi XOR în CNF “pur”

  • Encodările CNF pentru XOR pot fi mari și pot distruge propagarea.
  • Gauss incremental oferă propagare puternică și ieftină pentru circuite cu multe XOR-uri.

Interacțiune cu SAT

UBHNL folosește un model hibrid: 

CNF side (AND via Tseitin)  ⟷  XOR side (linear equations)
         unit propagation            Gauss propagation
         decisions/learning          implications/conflicts

Certificate / checking

  • Implicările XOR se pot verifica prin substituție în ecuații.
  • Pentru UNSAT global, UBHNL preferă certificate SAT-level (DRAT/LRAT) după reducere, sau obiecte de dovadă XOR-aware dacă există.

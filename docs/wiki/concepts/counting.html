#SAT / WMC / HashCount

Counting for Boolean theories: exact (KC) or approximate (hashing), with explicit audit metadata.

DRAFT: Wiki page.
Wiki Knowledge compilation Probabilistic DS-011

Definitions

  • #SAT: the number of models that satisfy a Boolean theory.
  • WMC (Weighted Model Counting): models with weights; the sum of model weights (a basis for discrete probabilistic inference).
  • HashCount: approximate counting via random hashing + SAT (reports (ε, δ) and seeds).

In UBHNL

  • Exact: prefer KC + equivalence verification.
  • Approximate: must include audit metadata (seeds, hash family, confidence parameters) and must be explicitly labeled as probabilistic.

Consequences

  • There is no simple “UNSAT certificate” analogue for #SAT; exactness requires checkable traces or composed verifications.
  • The orchestrator must model “trust”: approximate results are acceptable only if the user requests them (options/budget).

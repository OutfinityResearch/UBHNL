Model checking (Transition Systems)

Verificare de sisteme de tranziție: trace-uri ca witness și invariants ca dovezi.

Definiție

Un transition system este definit de: 

Init(s)      - condiția inițială
Trans(s,s')  - relația de tranziție
Safe(s)      - proprietatea de siguranță

Două rezultate tipice

  • DISPROVED: un trace s0 → s1 → ... → sk care pornește din Init și ajunge într-o stare nesigură.
  • PROVED: un invariant inductiv Inv(s) care implică Safe și e închis la tranziție.

Certificate (checking invariants)

Pentru un invariant Inv, checker-ul validează: 

1) Init(s) ⇒ Inv(s)
2) Inv(s) ∧ Trans(s,s') ⇒ Inv(s')
3) Inv(s) ⇒ Safe(s)

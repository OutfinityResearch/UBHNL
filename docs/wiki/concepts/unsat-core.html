UNSAT core

Subset de constrângeri care este deja nesatisfiabil (folosit pentru explain/debug).

Definiție

Un UNSAT core este un subset al constrângerilor C ⊆ T astfel încât C este UNSAT. În UBHNL, core-ul se raportează preferabil la origins (statement-level), nu la clauze interne.

“Minimal” vs “minimal-ish”

  • Minimal core: dacă scoți orice constrângere, devine SAT (costisitor).
  • Minimal-ish: un core mic obținut heuristic (suficient pentru explain în practică).

Rol în explain

  • Pentru inconsistențe: “aceste 3 reguli din fișierele X/Y sunt în conflict”.
  • Pentru entailment: core-ul pentru T ∧ ¬φ descrie “de ce φ e derivat”.

Legătura cu certificate

Un UNSAT core nu este dovada completă de UNSAT. Dovada vine dintr-un certificat checkable (ex. DRAT/LRAT). Core-ul e un artefact de explicabilitate.

Separation Logic / shape analysis

Reasoning about memory/heap properties: shape invariants as certificates, often reduced to CHC/SMT.

DRAFT: Wiki page.
Wiki CHC SMT Certificate DS-011

Definition

Separation Logic is a logic for reasoning about heap memory (pointers, data structures) that separates memory regions. Shape analysis derives invariants about heap structure.

In UBHNL

  • Treated as an optional fragment (Frag_Sep) with dedicated backends.
  • Typical integration: reduction to CHC/SMT plus invariant checking.

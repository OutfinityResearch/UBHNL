QBF (Quantified Boolean Formula)

Boolean logic with quantifiers over bits: ∀/∃ over Boolean variables.

DRAFT: Wiki page.
Wiki Fragment Certificate DS-011

Definition

A typical QBF is in prenex form: 

Q1 x1  Q2 x2  ...  Qn xn :  φ(x1..xn)

where each Qi is ∀ or ∃, and φ is a Boolean formula (often CNF).

Why it differs from SAT

  • SAT asks for a model (an assignment).
  • QBF asks for a strategy for existentials as a function of universals (Skolem functions / winning strategy).

Certificates

  • SAT: checkable strategies / Skolem functions.
  • UNSAT: Q-resolution-style proofs (or other accepted formats).

In UBHNL

  • QBF is a separate fragment (Frag_QBF) with dedicated backends (QCDCL/CEGAR).
  • Some instances can be bounded-reduced to SAT/UBH, but that is a tactic, not the default rule.

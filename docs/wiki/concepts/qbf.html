QBF (Quantified Boolean Formula)

Boolean logic cu cuantificatori peste biți: ∀/∃ peste variabile booleene.

DRAFT: pagină de Wiki.
Wiki Fragment Certificate DS-011

Definiție

Un QBF tipic este în formă prenex: 

Q1 x1  Q2 x2  ...  Qn xn :  φ(x1..xn)

unde fiecare Qi este ∀ sau ∃, iar φ este o formulă booleană (adesea CNF).

De ce e diferit de SAT

  • SAT cere un model (o atribuire).
  • QBF cere o strategie pentru existențiale în funcție de universale (Skolem functions / winning strategy).

Certificate

  • SAT: strategie/funcții Skolem verificabile.
  • UNSAT: certificate de tip Q-resolution (sau alte formate acceptate).

În UBHNL

  • QBF este un fragment separat (Frag_QBF), cu backends dedicate (QCDCL/CEGAR).
  • Unele instanțe pot fi reduse bounded la SAT/UBH, dar asta este o tactică, nu regula.

BMC (Bounded Model Checking)

Verify properties up to depth k: reduce TS to SAT/SMT and search for a trace.

DRAFT: Wiki page.
Wiki Model checking IC3/PDR Type lowering

Definition

BMC encodes an execution of length k as a formula: 

Init(s0) ∧ Trans(s0,s1) ∧ ... ∧ Trans(s{k-1},sk) ∧ ¬Safe(sk)

If the formula is SAT, you get a trace; if it is UNSAT, there is no counterexample of length ≤ k.

In UBHNL

  • It is the standard fallback when IC3/PDR does not produce an invariant within budget.
  • Lowering: TS/BV → UBH/SAT (bit-level), with DRAT/LRAT certificates for UNSAT at a fixed bound.

Gröbner bases (peste GF(2))

Metodă algebrică pentru sisteme de ecuații polinomiale; opțională în UBHNL pentru fragmente polinomiale.

DRAFT: pagină de Wiki.
Wiki GF(2) ANF DS-011

Definiție (intuiție)

O bază Gröbner este un set “normalizat” de polinoame care permite decizii algoritmice precum: reducerea unui polinom la o formă normală și testul de apartenență la ideal (implicare algebrică).

În UBHNL

  • Constrângerile UBH pot fi privite ca ecuații polinomiale peste GF(2) (XOR=+, AND=·).
  • Un backend Gröbner ar fi alternativ/complimentar la SAT/XOR pentru anumite clase de probleme.
  • Este “opțional”: sistemul rămâne coerent fără implementare Gröbner, deoarece SAT/XOR este ruta principală.

Certificate

Pentru o afirmație de tip “polinomul p este consecință a sistemului”, certificatul tipic este un traseu de reducere care arată că p reduce la 0 modulo baza Gröbner. Checker-ul validează pașii de reducere.

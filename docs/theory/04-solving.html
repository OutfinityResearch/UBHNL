1) solve()

solve() răspunde la întrebarea “există un model care satisface constrângerile?”. Rezultatul este SAT/UNSAT/UNKNOWN.

  • SAT: include un witness verificabil (model).
  • UNSAT: include un certificat verificabil.
  • UNKNOWN: buget depășit / certificat lipsă / metodă probabilistică neacceptată.

2) prove(phi)

prove(phi) se reduce la solve(T ∧ ¬phi). Dacă rezultatul e UNSAT (cu certificat), atunci phi este PROVED.

3) De ce SAT + XOR hibrid

AND-urile se encodează în CNF prin Tseitin, iar XOR-urile sunt ecuații liniare GF(2). De aici decizia SAT+XOR nativ.

4) Explain: unsat core și origins

  • Un unsat core (origins) ajută la explicații și debugging.
  • UNSAT este acceptat doar cu certificat; core-ul nu înlocuiește certificatul.

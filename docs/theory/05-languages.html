1) Lexicon = semnătura sistemului

Lexicon definește tipuri/domenii, constante și predicate. Fără el, CNL/DSL nu pot fi tipate robust.

2) CNL: NL fără ambiguități

  • CNL are gramatică mică și deterministă.
  • Cuantificatorii și conectivii sunt expliciți; pronumele/coreferența sunt respinse.

3) DSL: IR textual “subject-first” + core logic

  • DSL este formatul preferat pentru theory files.
  • Regula @/$/vocab (și “un singur @”) face parsing-ul simplu și reduce erorile.

4) Typing și erori utile

Politica de vocab strict (decizie) înseamnă: simbol necunoscut = eroare de încărcare (cu linie/col), nu “simbol nou”.

1) Propositional (UBH/SAT)

A classic entailment example: run it as GoalKind=Prove and accept only if T ∧ ¬φ is UNSAT with a certificate.

2) SMT(BV) and bit-blasting

For bitvectors (BV), the orchestrator can choose: native SMT (with a proof object) or lowering to UBH/SAT (bit-blast + DRAT/LRAT). Concepts: SMT and type lowering.

3) Transition systems (TS)

For TS safety, a counterexample is a checkable trace: concepts: model checking, IC3/PDR.

4) Hole filling

A query with holes returns witnesses as holeAssignments.
